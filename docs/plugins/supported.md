# Supported Plugins

The following plugins are bundled natively with the repository inside the `plugins/` directory.

## Docker (`plugins/docker`)
- **Capabilities Provided**: `runtime.docker`
- **Supported OS**: Windows, Linux, macOS.
- **Description**: Installs the Docker Engine and sets up appropriate daemon permissions.

## Git (`plugins/git`)
- **Capabilities Provided**: `runtime.git`
- **Supported OS**: Windows, Linux, macOS, Android (Termux).
- **Description**: Installs the Git version control system.

## Python (`plugins/python`)
- **Capabilities Provided**: `runtime.python`
- **Supported OS**: Windows, Linux, macOS.
- **Description**: Installs Python 3.x and Pip.

## Pocketbase (`plugins/pocketbase`)
- **Capabilities Provided**: `service.pocketbase`
- **Supported OS**: Windows, Linux, macOS.
- **Description**: Downloads and provisions the Pocketbase executable as a background service.
