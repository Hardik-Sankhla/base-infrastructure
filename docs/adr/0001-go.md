# ADR 001: Go Core Architecture

## Status
Accepted

## Context
We need a fast, cross-platform, single-binary foundation that avoids the fragility of complex shell scripts.

## Decision
The core framework will be written entirely in Go. Bash and PowerShell are strictly reserved for minimal entrypoint wrappers.

## Consequences
- Requires Go tooling for development.
- Ensures robust type-safety, strong concurrency, and easy cross-compilation.
