# Merge Checklist

Before merging a feature branch into the default branch, verify:

- [ ] Has the Reviewer persona signed off on the code and architectural boundaries?
- [ ] Are there zero failing CI workflows for the specific commit being merged?
- [ ] Is the branch fully up-to-date with `main`?
- [ ] Have all merge conflicts been resolved correctly?
- [ ] Has the `implementation_report.md` been reviewed?
